## ❌ If SMS Doesn't Arrive

**Check this in order:**

### 1️⃣ Check Semaphore Account
- Go to: https://semaphore.co
- Log in
- Check "SMS Balance" (should be > 0)
- Check "Account Status" (should be active)
- ✅ If numbers look good → Continue to #2

### 2️⃣ Check Your Phone Number
```bash
# Try with your phone number in different formats:

# Format 1: Standard (09XXXXXXXXX)
python manage.py test_sms --phone "09171234567"

# Format 2: International (+639XXXXXXXXX)
python manage.py test_sms --phone "+639171234567"

# Format 3: Just digits (639XXXXXXXXX)
python manage.py test_sms --phone "639171234567"

# Format 4: Without leading 0 (9XXXXXXXXX)
python manage.py test_sms --phone "9171234567"
```
- ✅ If works in any format → SMS is OK, move to #3

### 3️⃣ Check if Free Tier Whitelisting is Required
- Log into Semaphore: https://semaphore.co
- Look for "Whitelisted Numbers" or "Verified Numbers"
- If red warning: You need to verify/whitelist your phone number
- Add phone number to whitelist
- Try test command again
- ✅ If SMS arrives → SMS is now working!

### 4️⃣ Check SMSLog for Errors
- Go to: http://localhost:8000/admin/intake/smslog/
- Find your failed SMS (Status: "failed")
- Click it
- Look at "error_message" field
- Common errors:
  - `"401 Unauthorized"` → API key is wrong
  - `"402 Payment Required"` → Semaphore balance is 0
  - `"400 Bad Request"` → Phone number is malformed
